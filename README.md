# Telegram бот для поиска фильмов

Бот для поиска фильмов, использует API кинопоиска.

![PyPI - Python Version](https://img.shields.io/pypi/pyversions/pyTelegramBotAPI)

## Технологии

- pyTelegramBotAPI
- sqlite3
- loguru
- peewee
- dotenv

## Установка

1. Клонируйте или скачайте репозиторий с gitlab/github
2. Введите команду `docker compose up -d` из папки с проектом

## Настройка

Нужно создать файл .env в корне проекта.

Указать переменные окружения: telegram токен, ключ API

*Пример:*

```
BOT_TOKEN = "Ваш токен для бота, полученный от @BotFather"
KINOPOISK_API_KEY = "Ваш ключ полученный от API по адресу https://kinopoisk.dev/"
```

## Функционал

1) /start
    - Приветствует пользователя.
    - Вносит данные о пользователе в БД, создаётся дефолтный фильтр.
2) /help
    - Выводит справку о всех командах.
3) /history
    - Выводит историю последних 10 запросов.
    - Далее пользователь может выбрать номер запроса, просмотреть его детально.
4) /top_films
    - Предоставляет меню выбора из топ сериалов, фильмов и т.д.
5) /premiere
    - Пользователь выбирает месяц кинопремьер, далее выводится список.
6) search_filter
    - Поиск фильмов по уже настроенному фильтру, выбор сортировки, и количество фильмов.
7) filter
    - Настройка фильтра, рейтинг, год, жанр, тип (сериал, фильм, мультфильм).
8) current_filter
    - Просмотр текущего фильтра.
